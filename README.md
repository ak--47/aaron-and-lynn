# aaronandlynn.com

Static single-page site for **aaronandlynn.com**, served from GitHub Pages.

## Deploy

Push to `main` → the [Pages workflow](.github/workflows/pages.yml) publishes the repo root
via GitHub Actions. Custom domain is set in [`CNAME`](CNAME).

## Notes

- Single page: [`index.html`](index.html).
- Honeymoon gallery links out to Dropbox (too large to host here).
- Images were downscaled/optimized for web; full-resolution originals live in the
  Bluehost backup (`~/tmp/bluehostbackup`), not in this repo.
- The "Our Story" animation is an `.mp4` (converted from the original `.gif`).
